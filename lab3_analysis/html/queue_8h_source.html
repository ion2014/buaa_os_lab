4 /*
5  * This file defines three types of data structures: lists, tail queues,
6  * and circular queues.
7  *
8  * A list is headed by a single forward pointer(or an array of forward
9  * pointers for a hash table header). The elements are doubly linked
10  * so that an arbitrary element can be removed without a need to
11  * traverse the list. New elements can be added to the list before
12  * or after an existing element or at the head of the list. A list
13  * may only be traversed in the forward direction.
14  *
15  * A tail queue is headed by a pair of pointers, one to the head of the
16  * list and the other to the tail of the list. The elements are doubly
17  * linked so that an arbitrary element can be removed without a need to
18  * traverse the list. New elements can be added to the list before or
19  * after an existing element, at the head of the list, or at the end of
20  * the list. A tail queue may only be traversed in the forward direction.
21  *
22  * A circle queue is headed by a pair of pointers, one to the head of the
23  * list and the other to the tail of the list. The elements are doubly
24  * linked so that an arbitrary element can be removed without a need to
25  * traverse the list. New elements can be added to the list before or after
26  * an existing element, at the head of the list, or at the end of the list.
27  * A circle queue may be traversed in either direction, but has a more
28  * complex end of list detection.
29  *
30  * For details on the use of these macros, see the queue(3) manual page.
31  */
